# SuperSecret Mind Connection App

## Kurulum

1. Gerekli paketleri yükleyin:
```bash
npm install
```

2. Uygulamayı başlatın:
```bash
npm start
```

veya geliştirme modunda başlatmak için:
```bash
npm run dev
```

3. Tarayıcınızda şu adresi açın:
```
http://localhost:3000
```

## Özellikler

- Kullanıcı girişi
- Dashboard görünümü
- Zihinsel bağlantı kurma
- Gerçek zamanlı chat
- Kullanıcı durumu gösterimi (aktif, meşgul, çevrimdışı)
